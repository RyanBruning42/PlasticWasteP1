Context

This data set was independently gathered by an individual in order to contribute to the global audit that the Break Free from Plastic organization holds each year. They collected data in their own town of St. John in Newfoundland. The dataset has information on the amount of different plastic waste (polyester, PVC, polyethylene, etc…) as well as the grand total of all plastic types, number of counting events, and number of volunteers at each cleanup.

Data Cleaning:

From the included code on the data set, we can see that they have already taken measures to clean the data. It appears that most of the cleaning is making sure that each variable is the correct type as well as renaming columns and fixing all the numeric values.

Research Questions w/ Data:

  • Does the mean average of plastic waste per count event differ between each plastic type?

  • Does the overall plastic waste differ between 2019 and 2020?

Research Questions w/ Supplemental Data:

  • Do richer countries generate more plastic waste per person?

  • Did the COVID-19 lockdown in 2020 affect plastic waste pollution in the same way it affected CO2 emissions?

get_gdp <- function(iso2) {
  
  url <- glue("https://api.worldbank.org/v2/country/{iso2}/indicator/NY.GDP.MKTP.CD?format=json")
  
  my_data <- fromJSON(url)
  
  if(is.null(my_data[[2]])){
    return(NULL)
  } else{
    return(my_data[[2]] %>% 
             filter(date %in% c("2019","2020")) %>% 
             select(date, value) %>% 
             mutate(iso2c = iso2,
                    value = value/1e9) %>% 
             rename(gdp_billions = value))
  }
}

model_data <- meta_set %>%
  filter(year == 2019, parent_company == "Grand Total") %>%
  mutate(
    gdp_per_capita = gdp_billions * 1,000,000,000 / population,

model <- lm(plastic_waste_per_capita ~ gdp_per_capita, data = model_data)

model_summary <- summary(model)
